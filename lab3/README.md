# Лабораторная №3

В целях эффективности был разработан общий протокол по опыту разработки приложений для стандартизированных протоколов. Форматы данных, передаваемых общим протоколом, представлены в файле [shared-proto.md](shared-proto.md).

Все части лабораторной были выполнены на ЯП Python и объеденены модулями с общим кодом также, чтобы ускорить разработку и не переписывать уже реализованные идеи. Для работы программ необходимо иметь Python 3.8 в системе.

Данная лабораторная была поделена на две части:
- [news](news.md);
- [pays](pays.md).
